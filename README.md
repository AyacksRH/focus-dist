# Focus — публичная раздача сборок

Этот репозиторий нужен **только для установки/обновления** приложения Focus
через eSign (исходный код — в приватном репозитории).

## Как подключить (один раз)
1. Открой **eSign** на iPhone → вкладка **Sources** (Источники).
2. Нажми **+** и вставь URL:
   ```
   https://raw.githubusercontent.com/AyacksRH/focus-dist/main/source.json
   ```
3. Убедись, что в eSign выбран твой сертификат (.p12 + .mobileprovision) как
   подпись по умолчанию.

## Как обновляться
Открыл eSign → Sources → потянул обновить → у приложения Focus жмёшь **Update**.
eSign сам скачает новую версию и переподпишет твоим сертификатом на телефоне.
Сертификат никуда не уходит.
